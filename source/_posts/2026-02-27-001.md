## 官网地址

[UI/UX Pro](https://ui-ux-pro-max-skill.nextlevelbuilder.io/)

## 安装

```bash
npm install -g uipro-cli
```

## 

运行后会在当前目录下创建一个`.cursor`目录,这个就是给 AI 的技能了

```bash
uipro init --ai cursor
```
