## 进入插件目录

ComfyUI 的插件目录默认是在项目根目录下的 `custom_nodes` 目录,所以你需要进入这个目录。

```bash
cd ComfyUI/custom_nodes
```

## 克隆仓库

前往`Github`将仓库克隆下来,如果是国内线路,则继续使用代理就行,如何使用请阅读`Linux下部署ComfyUI`这篇文章。

仓库地址: https://github.com/Comfy-Org/ComfyUI-Manager

```bash
git clone https://github.com/Comfy-Org/ComfyUI-Manager.git
```

## 安装依赖

进入仓库目录,安装依赖:

```bash
cd ComfyUI-Manager
pip install -r requirements.txt
```

## 重启服务

依赖安装完成后,把你正在运行的 ComfyUI 服务重启一下就行,如果不出意外,你访问 WEB 的话,应该能看见 Manager 的按钮了
