## 创建并切换分支

```bash
git switch -c dev/2025-12-31-dev
```

`switch`命令是切换分支的命令,`-c`选项是创建分支的选项,`dev/2025-12-31-dev`是分支名称。

## 查看分支

```bash
git branch
```

`branch`命令是查看分支的命令,会列出所有分支,当前分支会用`*`标记。


## 强制同步远程仓库

```bash
git fetch origin
git reset --hard origin/main
```

比较危险,不建议使用,除非你真的知道你在做什么。

执行后会丢弃本地所有修改,并强制与远程仓库同步,用于你想让本地和远程仓库一致时使用。

## 推送分支

```bash
git push origin dev/2025-12-31-dev
```

`push`命令是推送分支的命令,`origin`是远程仓库的名称,`dev/2025-12-31-dev`是分支名称。
