Minecraft 常用指令汇总

在《我的世界》中，合理使用指令（Command）可以极大地提升测试效率或改善生存体验。在使用以下指令前，请确保地图已开启“允许作弊”。

🎮 模式切换 (Game Mode)

最基础的操作，用于在不同游戏体验间快速切换。

指令 简写 (旧版本) 效果
/gamemode creative /gamemode 1 创造模式：无限资源、飞行、无敌。
/gamemode survival /gamemode 0 生存模式：常规玩法，受伤害和饥饿限制。
/gamemode adventure /gamemode 2 冒险模式：无法破坏方块，常用于解谜地图。
/gamemode spectator /gamemode 3 旁观模式：穿墙飞行，无法与世界交互。

小技巧：在 Java 版 1.16+ 中，长按 F3 + F4 可以快速呼出模式切换菜单。

🛡️ 核心规则管理 (Game Rule)

通过修改游戏规则（Gamerule）来改变世界运行逻辑。

  • 死亡不掉落 (最常用)：
    /gamerule keepInventory true
  • 禁用生物破坏 (防止苦力怕炸坑、末影人搬砖)：
    /gamerule mobGriefing false
  • 禁用火焰蔓延：
    /gamerule doFireTick false
  • 停止昼夜交替 (冻结时间)：
    /gamerule doDaylightCycle false

🌩️ 环境与时间控制

掌控世界的天气与阴晴。

时间控制

  • 设为白天：/time set day
  • 设为夜晚：/time set night
  • 查询当前游戏天数：/time query day
  • 增加时间：/time add <数值>

天气控制

  • 晴天：/weather clear
  • 雨天：/weather rain
  • 雷雨：/weather thunder
  • 锁定天气时长：/weather clear 999999 (让天空长时间保持晴朗)

⚔️ 生物与玩家操作

用于清理怪物或定位坐标。

清理生物 (Kill)

  • 杀死所有生物 (慎用，包括你自己)：
    /kill @e
  • 只杀死附近的敌对生物 (Java版常用技巧)：
    /kill @e[type=!player,type=!item,type=!minecart]
  • 清除掉落物 (缓解卡顿)：
    /kill @e[type=item]
  • 杀死特定生物 (如苦力怕)：
    /kill @e[type=creeper]

传送与位置 (TP)

  • 传送到指定坐标：
    /tp <x> <y> <z>
  • 将 A 玩家传送到 B 玩家处：
    /tp <玩家A> <玩家B>
  • 寻找最近的结构 (如村庄、要塞)：
    /locate structure village (新版本) / /locate village (旧版本)

💎 物品与经验给予

  • 获得特定物品：
    /give @s diamond_sword 1 (给自己一把钻石剑)
  • 增加经验等级：
    /xp add @s 100 levels (给自己增加 100 级经验)
  • 清除背包所有物品：
    /clear

💡 指令目标选择器说明

在输入指令时，经常会看到 @ 开头的符号，它们的含义如下：

  • @s：执行指令的本人（你自己）。
  • @p：距离最近的玩家。
  • @a：所有玩家。
  • @e：所有实体（包括生物、掉落物、箭矢等）。
  • @r：随机一名玩家。

⚠️ 注意事项

  1. 大小写敏感：所有指令必须使用英文小写。
  2. Tab 键补全：在输入指令时，多按 Tab 键。系统会自动提示你接下来的参数，这能极大地避免语法错误。
  3. 极限模式说明：极限模式无法直接通过 /gamemode 切回生存，因为极限模式在存档创建时就已经锁定。如果死后想复活，需要“对局域网开放”并“开启作弊”。
